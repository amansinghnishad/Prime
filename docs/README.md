# PrimeTrade (MERN Stack)

A full-stack productivity dashboard built using the MERN stack that allows users to manage **notes** and **tasks** efficiently with authentication, protected routes, and a clean, scalable architecture.

## Features
- User authentication (JWT + cookies)
- Protected routes
- Notes management (create, list)
- Tasks management (create, list, update status)
- Optimistic UI updates
- Responsive dashboard layout
- Modular and scalable architecture

## Tech Stack
**Frontend**
- React
- React Router
- Tailwind CSS
- Axios

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Getting Started

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
npm install
npm run dev
```

## Project Philosophy

This project follows industry-standard separation of concerns, keeping UI, business logic, and API communication clearly isolated.
